= ZAQRIT
A repository for ZAQRIT.

= Dependency
* VuePress

= Setup
* Clone and install this project.
```bash
git clone https://github.com/rit-rit/zaqrit.git
cd zaqrit
npm install
```

= Usage
* Run build command
```bash
npm run build
```
